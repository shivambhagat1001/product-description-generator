# 🧠 Product Description Generator

A Python automation script that generates **SEO-friendly product descriptions, FAQs, and short descriptions** from CSV data.

---

## 🚀 Features

- 📄 Reads product data from CSV
- ✍️ Generates dynamic product descriptions
- ❓ Auto-generates FAQs
- 🔁 Uses multiple templates to avoid duplicate content
- 📤 Exports updated CSV for bulk upload

---

## 🛠 Tech Stack

- Python
- Pandas

---

## 📂 Input

CSV file with product details such as:
- Brand
- Size
- Year
- Category
- Price

---

## ⚙️ How It Works

1. Reads input CSV file  
2. Extracts product details  
3. Generates:
   - Description (HTML format)
   - FAQ section
   - Short description  
4. Writes updated data to output CSV  

---

## ▶️ How to Run

```bash
pip install pandas
python script.py
```
